# Varsity Sports Performance Tracker 

A lightweight, interactive Python data script built to parse, calculate, and log dynamic player scoring data arrays. Designed to showcase foundational computational logic, terminal input validation, and functional math mapping.

##  Concepts Demonstrated
* Dynamic User-Input Parameters
* Numerical Type Casting & Array Management
* Algorithmic Data Optimization (Averages & Peak Tracking)

##  Implementation Commands
Execute the script through your native system terminal loop:
```bash
python tracker.py
```
